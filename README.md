# Discover the movie of your birthdate

Built with AngularJS and using [The Movie DB API].

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Tests

```bash
npm test
```

[The Movie DB API]: <https://www.themoviedb.org/documentation/api>
